# Task Description

Use React DevTools browser extension. Install and open the Components tab. Inspect the component tree hierarchy. View and edit props in real-time. View state and hooks for functional components. Use the Profiler tab to record performance: identify components causing slow renders, view render duration, find unnecessary re-renders. Use the ""Highlight updates when components render"" feature to visualize re-renders. Demonstrate finding a performance issue and optimizing with React.memo or useMemo.
