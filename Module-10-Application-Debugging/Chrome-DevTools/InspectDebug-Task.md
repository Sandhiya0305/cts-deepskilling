# Task Description

Use Chrome DevTools for React debugging. Open DevTools (F12). Elements tab: inspect DOM tree, modify styles live, test responsive layouts. Console tab: log variables, run JavaScript, view error stack traces. Sources tab: set breakpoints in JavaScript, step through code, view call stack. Network tab: monitor API requests, inspect request/response headers and bodies, simulate slow network. React DevTools extension: inspect component tree, view props and state, identify unnecessary re-renders with Profiler.
