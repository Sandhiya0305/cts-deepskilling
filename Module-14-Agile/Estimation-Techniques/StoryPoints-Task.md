# Task Description

Document Agile estimation techniques. Story points: relative sizing based on complexity, effort, and uncertainty (not time). Fibonacci sequence (1, 2, 3, 5, 8, 13, 21). Planning Poker: team discusses, each member selects card, reveal simultaneously, discuss outliers, re-estimate if needed. Velocity: average story points completed per sprint, used for future capacity planning. Burndown chart: visual tracking of remaining work during sprint. Include a sample sprint with tasks, story points, and burndown calculation.
