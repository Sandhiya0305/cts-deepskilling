# Task Description

Write 5 user stories following the INVEST principle (Independent, Negotiable, Valuable, Estimable, Small, Testable). Format: ""As a [role], I want [goal], so that [benefit]"". Stories: 1) Employee registration with email verification, 2) Admin viewing employee reports with filters, 3) User resetting forgotten password, 4) Manager approving leave requests, 5) System sending monthly salary slips. For each story, write acceptance criteria in Given-When-Then format. Include story point estimation (1, 2, 3, 5, 8) with rationale.
