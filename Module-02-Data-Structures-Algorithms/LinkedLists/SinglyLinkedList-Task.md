# Task Description

Implement a Singly Linked List from scratch without using java.util.LinkedList. Create a Node inner class with data and next fields. Implement: insertAtHead, insertAtTail, insertAtPosition, deleteByValue, deleteAtPosition, search (return boolean or index), display (print all elements), getLength, and isEmpty. Include time complexity comments for each operation. Handle edge cases: empty list, single node, position beyond list length. Write a main method demonstrating all operations.
