# Task Description

Implement a Doubly Linked List from scratch with prev and next pointers in the Node class. Implement: insertAtHead, insertAtTail, insertAtPosition, deleteByValue, displayForward, displayBackward, and getLength. Show the advantage over Singly Linked List for reverse traversal and deletion operations. Include time complexity comments. Handle edge cases. Write a main method that demonstrates insertion at both ends, forward display, backward display, and deletion.
