# Task Description

Implement core array operations from scratch without using Java Collections framework. Include: traversal (print all elements), insertion at beginning/middle/end, deletion by value and by index, searching using linear search and binary search (on sorted array). For each operation, add comments specifying time complexity (O(1), O(n), O(log n)) and space complexity. Include edge case handling: empty array, full array, index out of bounds, duplicate values. Write a main method to demonstrate each operation with sample data.
