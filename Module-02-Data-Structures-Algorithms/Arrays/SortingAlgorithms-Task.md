# Task Description

Implement the following sorting algorithms from scratch: Bubble Sort, Insertion Sort, Selection Sort, Merge Sort, Quick Sort. For each algorithm, include: step-by-step explanation in comments, the implementation, best/average/worst case time complexity analysis, and space complexity. Include a main method that generates a random array, sorts it using each algorithm, and prints the before/after state. Compare the performance of O(n^2) algorithms vs O(n log n) algorithms on larger arrays.
