# Task Description

Create a basic Spring MVC controller. Use @Controller and @RequestMapping(""/hello""). Create methods: showForm() returning a view name for input form, processForm(@RequestParam String name, Model model) that adds the name to model and returns result view. Create corresponding JSP or Thymeleaf templates (hello-form.html and hello-result.html). Show how Spring MVC handles request mapping, parameter binding, and view resolution. Include form validation with @Valid and BindingResult.
