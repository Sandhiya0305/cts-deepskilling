# Task Description

Document and demonstrate a Feature Branch workflow. Create a main branch. Create feature/login branch from main. Make commits on feature branch. Push feature branch to remote. Create a Pull Request. Review code, address comments. Merge feature branch to main using merge commit (or squash merge, or rebase). Delete feature branch after merge. Show how to handle merge conflicts: identify conflicted files, resolve markers, add resolved files, complete merge. Include diagrams of branch history before and after merge.
