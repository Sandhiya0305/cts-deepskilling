# Task Description

Create a Java project demonstrating Lombok annotations. Include: @Data on a POJO to auto-generate getters, setters, toString, equals, hashCode, @NoArgsConstructor, @AllArgsConstructor, @RequiredArgsConstructor for constructors, @Builder for fluent object creation, @Slf4j to auto-inject a logger field. Show the difference between a verbose Java class (50+ lines) and the same class with Lombok annotations (10 lines). Include a main method that uses the builder pattern to create objects and logs messages using the auto-generated logger.
