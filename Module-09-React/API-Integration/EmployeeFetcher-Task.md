# Task Description

Create a component that fetches employee data from a Spring Boot REST API. Use useEffect with empty dependency array for initial load. Implement loading state (spinner), error state (error message), and empty state. Use fetch API and also show equivalent with axios (with interceptors for auth headers). Display data in a sortable, filterable table. Implement pagination with previous/next buttons. Add a refresh button to re-fetch data. Handle network errors and 401/403 responses.
