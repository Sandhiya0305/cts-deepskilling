# Task Description

Set up React Router for a multi-page SPA. Install react-router-dom. Create routes: / (Home), /employees (EmployeeList), /employees/:id (EmployeeDetail), /employees/new (AddEmployee), /about (About). Use BrowserRouter, Routes, Route, Link, NavLink components. Implement nested routes if needed. Show useParams for accessing URL parameters, useNavigate for programmatic navigation, useLocation for query parameters. Include a 404 Not Found route. Demonstrate protected routes that redirect to login if not authenticated.
