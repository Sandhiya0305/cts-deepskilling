# Task Description

Create a controlled registration form with validation. Fields: username (required, 3-20 chars), email (required, valid format), password (required, 8+ chars, 1 uppercase, 1 number, 1 special char), confirmPassword (must match password), agreeToTerms (checkbox, required). Use useState for form data and errors. Validate on submit and show inline error messages. Disable submit button until all fields are valid. Show password strength indicator. Handle form submission with fetch/axios POST request.
