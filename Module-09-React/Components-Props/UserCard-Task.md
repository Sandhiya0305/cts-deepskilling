# Task Description

Create a reusable UserCard functional component. Props: name (string, required), email (string, required), role (string, default ""User""), avatarUrl (string, optional), isActive (boolean, default true). Use prop-types for runtime type checking or TypeScript interfaces. Show default props using default parameters. Create a parent UsersList component that renders multiple UserCards with different data. Demonstrate prop drilling and discuss when it becomes problematic (leading to Context API or Redux).
