# Task Description

Implement the Decorator pattern. Create a Coffee interface with getCost() and getDescription() methods. Create a SimpleCoffee implementation. Create abstract decorators: MilkDecorator, SugarDecorator, WhipDecorator that wrap a Coffee object and add to its cost and description. Demonstrate ordering a "Dark Roast with Milk and Sugar" by stacking decorators dynamically. Show how decorators add responsibilities at runtime without subclassing, and how order of decoration matters.
