# Task Description

Implement the Open/Closed Principle (OCP). Create a Shape area calculator. In the BEFORE version, use if-else or switch statements to calculate area for different shapes (Circle, Rectangle, Triangle). In the AFTER version, create an abstract Shape class with an abstract area() method, and have each shape subclass implement its own area calculation. Demonstrate that new shapes can be added without modifying existing code. Include comments on why OCP matters for extensibility.
