# Task Description

Create a REST controller demonstrating various parameter types. Use @PathVariable for resource IDs (/employees/{id}), @RequestParam for optional filters (/employees?dept=IT&minSalary=50000), @RequestBody for JSON payloads in POST/PUT, @RequestHeader for reading custom headers, @CookieValue for reading cookies. Show default values with @RequestParam(defaultValue=""0""). Demonstrate binding query parameters to a POJO using @ModelAttribute or @ParameterObject. Include validation annotations on parameters.
