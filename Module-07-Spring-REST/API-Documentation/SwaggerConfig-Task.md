# Task Description

Configure SpringDoc OpenAPI for automatic API documentation. Add springdoc-openapi-starter-webmvc-ui dependency. Create @OpenAPIDefinition with Info (title, version, description, contact), Servers, and SecurityScheme (Bearer JWT). Annotate controller methods with @Operation(summary, description) and @ApiResponse for each status code. Access Swagger UI at /swagger-ui.html. Show how annotations improve developer experience and reduce onboarding time for API consumers.
