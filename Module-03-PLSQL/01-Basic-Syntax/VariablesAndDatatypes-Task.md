# Task Description

Write PL/SQL anonymous blocks demonstrating variable declaration and usage. Include: declaring variables with explicit datatypes (NUMBER, VARCHAR2, DATE, BOOLEAN), using %TYPE to inherit datatype from a table column, using %ROWTYPE to declare a record matching a table row structure. Show assignment using := operator. Use DBMS_OUTPUT.PUT_LINE to print variable values. Include comments explaining when to use %TYPE and %ROWTYPE for maintainability.
