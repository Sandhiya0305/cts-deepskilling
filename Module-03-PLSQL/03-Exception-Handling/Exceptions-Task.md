# Task Description

Write PL/SQL blocks demonstrating exception handling. Include: predefined Oracle exceptions (NO_DATA_FOUND, TOO_MANY_ROWS, ZERO_DIVIDE, DUP_VAL_ON_INDEX), user-defined exceptions declared in DECLARE section and raised with RAISE, RAISE_APPLICATION_ERROR for custom error codes and messages. Show exception handlers with WHEN specific_exception THEN, WHEN OTHERS THEN, and using SQLCODE and SQLERRM to get error details. Create a salary validation block that raises an exception if salary is negative or exceeds a maximum limit.
