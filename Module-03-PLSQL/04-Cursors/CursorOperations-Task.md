# Task Description

Write PL/SQL demonstrating all cursor types. Include: implicit cursor (SELECT INTO with row count check using SQL%ROWCOUNT), explicit cursor (DECLARE cursor_name CURSOR FOR select_statement, OPEN, FETCH into variables, CLOSE), cursor FOR loop (simpler syntax), parameterized cursor (accepting input parameters), and REF CURSOR (strongly typed cursor variable for dynamic SQL). Create a practical example that fetches all employees from a department and prints their names, salaries, and hire dates using different cursor approaches.
