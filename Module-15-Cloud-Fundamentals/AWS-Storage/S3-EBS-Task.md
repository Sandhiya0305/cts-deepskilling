# Task Description

Document AWS storage services. S3 (Simple Storage Service): buckets (globally unique names), objects (files with metadata), storage classes (Standard, Intelligent-Tiering, Standard-IA, Glacier for archive). Create bucket, upload file, set permissions (public vs private), enable versioning and lifecycle policies. EBS (Elastic Block Store): attach volumes to EC2, types (gp2, gp3, io1, io2), snapshots for backup. Compare S3 (object storage, HTTP access) vs EBS (block storage, attached to single EC2). Include pricing considerations.
