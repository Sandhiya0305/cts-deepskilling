# Task Description

Document AWS database services. RDS (Relational Database Service): create MySQL/PostgreSQL instance, configure Multi-AZ for high availability, automated backups, read replicas for scaling. Connect from application using endpoint URL. DynamoDB: NoSQL key-value and document database, create table with Partition Key and optional Sort Key, basic PutItem, GetItem, Query, Scan operations. Compare RDS (SQL, ACID, complex queries) vs DynamoDB (NoSQL, single-digit ms latency, massive scale). Include when to choose each based on data structure and access patterns.
